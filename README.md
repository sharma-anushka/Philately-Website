# **Indian Philately Hub**

## **Overview**
**Indian Philately Hub** is a comprehensive platform designed for philatelists across India to explore, order, and learn about philatelic materials issued by various postal circles. It aims to bring the vibrant world of philately online, creating a centralized space for enthusiasts to connect with the diverse postal history and culture of India.

## **Features**
- **Centralized Philatelic Information**: Access to philatelic materials released by all 23 postal circles in India.
- **Online Ordering**: Order stamps, first-day covers, and other philatelic products seamlessly.
- **Payment Integration**: Secure online payment system for philatelic purchases.
- **Cancellation Releases**: View details of cancellation releases from all postal circles.
- **Standard Postal Services**: Integrates with standard postal services for added convenience.

## **Tech Stack**
- **Frontend**: HTML, CSS, JavaScript (with responsive design).
- **Backend**: Node.js, Express.js.
- **Database**: MongoDB for managing ancillary data and user information.
- **Others**: RESTful APIs for data access and integration.
